# When
## SYNOPSIS
Specifies when one or more notifiers should be executed.
## SYNTAX

```
When [-Condition] <String> [<CommonParameters>]
```

## DESCRIPTION
Specifies when one or more notifiers should be executed.
## EXAMPLES

### Example 1
```
WatchmenOptions {
    notifies {
        when 'Always'
    }
}
```

Tells Watchmen to execute all notifiers regardless if they pass or fail.
## PARAMETERS
